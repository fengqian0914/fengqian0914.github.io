## 作品集

### APP 類

#### 🥇 捷旅印然（畢業專題）
`Kotlin` `Firebase` `API`

Android APP。以 Android Kotlin 進行設計與開發，使用 API 串接與 Firebase 雲端服務，內建 QR Code 掃描功能，便利使用者於捷運站即時記錄站點並累積成就點數。

🎬 [YouTube](https://www.youtube.com/watch?v=WiQyS0wx8Bg) ｜ 💻 [GitHub](https://github.com/fengqian0914/Metro-Journey-Stamp-Collection)

---

#### 🚧 RouteEats — 路吃（開發中）
`Flutter` `Firebase` `Claude` `VibeCoding`

AI VibeCoding 專案。一款以「路途」為核心的餐廳探索雙平台 App。不同於傳統「找附近的餐廳」，RouteEats 讓你在從 A 到 B 的移動途中，找到順路可以停靠的餐廳，支援新增中途停靠點、規劃多段路線與一鍵導航。採用 Flutter 3.x 跨平台開發，整合 Google Maps、Directions API、Places Nearby Search API 與 Firebase Auth + Cloud Firestore。

---

#### API 整合系統
`Kotlin` `API` `OpenData`

💻 [GitHub](https://github.com/fengqian0914/fengqian0914.github.io/tree/main/Android_App/opendataAPI)
